Winks refers to the personalized content of the opposite side that is displayed on your handset terminal when a phone talk event occurs. Use the UP and DOWN keys on handset keypad to move the cursor to the setup item. Press OK key to confirm the item. Click BACK key to go back to the previous interface. If the consumer unloads Winks software，resemble it again installation or
cover installation，in order to ensure that Winks software regularity works，please starting it at first.In the process of activation，user must choose whether agreeing a short message to send out，if user choose no，user can not active Winks software. Please choose Permissions－messaging－choose any options except no manually. Then user could active Winks software again.
