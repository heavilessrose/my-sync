Detailed Description

Base class for AABB tree.

AABB trees are trees where each node is bounded by an axis aligned box and leafs contain one or more primitives. The type of the primitive depends of the tree implementation (this just a base class with no implementation). OZCollide provides built-in implementations: AABBTreePoly (AABB on nodes, polygons on leafs), AABBTreeSphere (AABB on nodes, spheres on leafs) and AABBTreeAABB (AABB on nodes, AABB on leafs). You can of course add your own implementation. Note about the memory's layout: all nodes are placed in one big block. The pointer to this block is the address of the root's node. All leafs are also placed in one big block. In other words, there are two pointers, one for the nodes array, one for the leafs array: everything is consecutive in memory.
