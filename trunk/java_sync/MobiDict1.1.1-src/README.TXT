To compile the application with Sun J2ME Wireless Toolkit:

1. Create application with name MobiDict from the KToolbar with MIDlet Class Name dict.Mobidict
2. Copy the content of the /src folder to <wireless toolkit folder>/apps/MobiDict/src
3. Copy the content of the /res folder to <wireless toolkit folder>/apps/MobiDict/res
4. Compile with Package->Create Package or with Package->Create Obfuscated Package if you configured ProGuard with the J2ME Toolkit
