Style Guide For C++:
--------------------

U = upper case letters
x = lower case letters for rest of word,
    otherwise uppercase letter for the first letter of each word

Type Names:
-----------
Uxxxxx

Function Names:
---------------
Uxxxxx

Data Members of Classes:
------------------------
_Uxxxxx

Variables, Arguments and Parameters:
------------------------------------
Uxxxxx_

Constants, defines and enumerated types:
----------------------------------------
UUUUU
