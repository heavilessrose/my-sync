Builds an AABBTree constitued of AABBs.

This method allows to automatically build a tree based on a list of AABBs. The built tree is immediatly available for use and save.

Parameters:
_nbBoxes [in] The number of AABBs pointed by _boxes.
_boxes [in] The list of AABBs to build a tree with.
_leafDepth [in] The maximum number of AABBs hold by a leaf.
_monitor [in] Optional, a Monitor instance to obtain informations during the build.
Returns:
The built tree.
