Builds an AABBTree constitued of spheres.

This method allows to automatically build a tree based on a list of spheres. The built tree is immediatly available for use and save.

Parameters:
_nbSpheres [in] The number of spheres pointed by _spheres.
_spheres [in] The list of spheres to build a tree with.
_leafDepth [in] The maximum number of spheres hold by a leaf.
_monitor [in] Optional, a Monitor instance to obtain informations during the build.
Returns:
The built tree.
